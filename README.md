# fish-check-v4
【2026原创】鱼类识别系统~Python+深度学习+卷积神经网络算法+模型训练+计算机毕设项目

## 一、介绍
鱼类识别系统（['墨鱼', '多宝鱼', '带鱼', '石斑鱼', '秋刀鱼', '章鱼', '红鱼', '罗非鱼', '胖头鱼', '草鱼', '银鱼', '青鱼', '马头鱼', '鱿鱼', '鲇鱼', '鲈鱼', '鲍鱼', '鲑鱼', '鲢鱼', '鲤鱼', '鲫鱼', '鲳鱼', '鲷鱼', '鲽鱼', '鳊鱼', '鳗鱼', '黄鱼', '黄鳝', '黑鱼', '龙头鱼']），技术栈如下：
● 前端：Vue3+Element plus
● 后端：Flask框架
● 算法：卷积神经网络算法（resnet50）、TensorFlow
关键技术：深度学习、模型训练、图像识别、人工智能、Python、CNN卷积网络
